# Coconut

Coconut is a JVM-level minecraft antimalware. It runs as a JVMTI Agent & hooks java functions, just like an [EDR](https://en.wikipedia.org/wiki/Endpoint_detection_and_response)

> [!CAUTION]
> This project is a very early proof of concept, it is unstable and shouldn't be used other than for testing purposes.
